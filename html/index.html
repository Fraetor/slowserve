Slow Webpage

This page is being served very slowly by Slowserve.

You can change the rate at which it is served by adding a query string to the URL, e.g.: http://example.com/?rate=300. This rate is in bytes per second. By default this page is served at 100 bytes per second, which is roughly equivalent to common modems in the mid 1970s.

Loading an Image

We can even load images. Slowly...

Fortunately JPEG has a handy feature called progressive loading. This is why you should be able to see a blurry preview well before the image has fully loaded. This is a feature that JPEG XL has further developed so that the important parts of the image can be loaded first.
